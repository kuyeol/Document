주요 그리스 문자 시각화

수학과 과학의 여러 분야에서 변수, 상수, 연산 등을 나타내기 위해 사용되는 주요 그리스 문자를 시각 자료와 함께 정리했습니다.

1. 파이 (Π, π) - 원주율

원은 크기에 상관없이 원주(둘레)를 지름으로 나눈 비율이 항상 일정합니다.

파이(π)는 원의 원주(둘레)와 지름 사이의 비율을 나타내는 수학 상수입니다. 이 값은 어떤 원에서든 약 3.14159로 항상 동일하며, 수학에서 가장 중요한 상수 중 하나입니다.

2. 시그마 (Σ, σ) - 합산

대문자 시그마(Σ)는 주어진 범위 내의 모든 항을 더하라는 의미의 연산 기호입니다.

대문자 시그마(Σ)는 수학에서 합산(Summation)을 나타내는 기호입니다. 시그마 아래에 있는 시작 값(i=1)부터 위에 있는 끝 값(n)까지, 뒤따르는 항(xᵢ)의 값을 순서대로 모두 더하라는 의미입니다.

3. 델타 (Δ, δ) - 변화량

대문자 델타(Δ)는 두 지점 사이의 유한한 차이 또는 변화량을 의미합니다.

대문자 델타(Δ)는 변수의 변화량(Difference)을 나타냅니다. Δx는 x의 변화량, Δy는 y의 변화량을 의미하며, 주로 '나중 값 - 처음 값'으로 계산됩니다. 직선의 기울기는 Δy / Δx로 표현할 수 있습니다.

4. 기타 주요 그리스 문자 (참조표)

기호 (대/소) 이름 주요 용도
Α / α 알파 각도, 계수, 상수, 유의 수준(통계)
Β / β 베타 각도, 계수, 상수
Ε / ε 엡실론 극한에서 0에 가까운 아주 작은 양수, 오차
Θ / θ 세타 기하학에서의 각도
Λ / λ 람다 고유값(선형대수), 파장(물리)
Μ / μ 뮤 모평균(통계), 마이크로(단위 접두어)
Σ / σ 시그마 Σ: 합산 (Summation)
σ: 모표준편차(통계)
Ω / ω 오메가 Ω: 표본 공간(확률), 옴(전기 저항 단위)
ω: 각속도(물리)
